    json模块主要有四个比较重要的函数，分别是：
    1，dump - 将Python对象按照JSON格式序列化到文件中
    2，dumps - 将Python对象处理成JSON格式的字符串
    3，load - 将文件中的JSON数据反序列化成对象
    4，loads - 将字符串的内容反序列化成Python对象
